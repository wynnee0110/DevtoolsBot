# DevTools Telegram Bot

Small Telegram bot for system and developer tools.

## 🚀 Features

- 🌐 **Ping** — Ping websites
- 🔐 **Password** — Generate secure passwords
- 📜 **Lorem** — Generate lorem ipsum
- 🎨 **QR** — Generate QR codes
- 🔏 **Hash** — Hash text (MD5, SHA256, etc.)
- 🔤 **Base64** — Encode / decode
- 🌍 **IP Lookup** — Get geolocation info

## 🔗 Try my bot @Devtoolsbywayne_bot https://t.me/Devtoolsbywayne_bot

## 🛠 Installation

```bash
python -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt
```

## ⚙️ Configuration

Create a `.env` file:

```
BOT_TOKEN=your_telegram_bot_token_here
```

## 🚀 Run

```bash
python app/main.py
```
